# Java FSE Week 1 Exercises

This repository documents the comprehensive exercises completed during the Java Full Stack Engineering (FSE) training program, covering design patterns, algorithms, database programming, TDD, and logging frameworks.

## Core Modules & Objectives

### 1. Design Principles and Patterns
- **Objective**: Implement core design patterns to improve code maintainability and reusability.

### 2. Data Structures and Algorithms
- **Objective**: Solve complex business logic problems using efficient data structure implementations.

### 3. PL/SQL Programming
- **Objective**: Develop database-level logic using stored procedures and control structures.

### 4. TDD using JUnit5 and Mockito
- **Objective**: Master Test-Driven Development (TDD) and dependency isolation.
- **Key Focus**: Setup, assertions, test fixtures, stubbing, and interaction verification.

### 5. SLF4J Logging Framework
- **Objective**: Implement robust application logging to track execution and error reporting.
